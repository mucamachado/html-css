Os principais tipos de formatações

Negrito / Destaque / Bold

Nesse texto, temos um termo em negrito/bold utilizando a tag <b> *Não semântico

Já nesta frase, temos um termo em destaque utilizando a tag <strong> *Semântico

Itálico / Ênfase

Nessa frase, temos um termo em itálico utilizando a tag <i> *Não semântico

Já nessa frase, temos um termo com ênfase, utilizando a tag <em> *Semântico

Texto marcado

É possível também, criar um texto marcado utilizando a tag <mark>

Nesse parágrafo, também temos um texto marcado

Textos grandes e pequenos

Irei criar um texto grande e um texto pequeno nesse parágrafo

Texto deletado

Podemmos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois

Existe também o texto sublinhado que é feito com a tag <u> *Não semântica
